vue-router

Vue router 是 Vue.js 官方的路由管理器。它和 Vue 的核心深度集成，让构建单页面应用易如反掌

核心步骤

目标

  • 实现一个插件
    • 实现 VueRouter 类
      • 处理路由选项
      • 监控 url 变化，hashchange
      • 响应这个变化
    • 实现 install 方法
      • $router注册（this.$router.push）
      • 两个全局组件（router-link,router-view）
