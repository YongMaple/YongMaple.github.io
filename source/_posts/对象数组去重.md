reduce 除了用来求和之外

```js
const sum = arr.reduce((total, n) => total + n.count, 0)
```

还可以用来给对象数组去重

```js
let obj = {}
list = list.reduce((item, next) => {
  if (!obj[next.name]) {
    obj[next.name] = true
    item.push(next)
  }
  return item
}, [])
```
