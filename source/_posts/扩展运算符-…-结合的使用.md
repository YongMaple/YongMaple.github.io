引入mapActions
```javascript
import { mapActions, mapGetters } from 'vuex'
```
在`methods`中写
```javascript
...mapActions([
  'SET_MAIL_LIST'
]),
```
数组中的所有方法会被引入，在使用时只要
```javascript
this.SET_MAIL_LIST({mailList})
```
